## 🧠 Tentang PHP

PHP (PHP: Hypertext Preprocessor) adalah bahasa scripting **server-side** yang dijalankan di server. Hasilnya (HTML) dikirim ke browser. Dipelajari sejak kelas XI SMK RPL, diperdalam saat kuliah S1 Informatika.

```
Browser → Request → Server (PHP dijalankan) → HTML → Browser
```
