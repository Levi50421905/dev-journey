5. Properti Flex Item

flex-grow — Ambil Ruang Sisa

Menentukan berapa bagian ruang sisa yang diambil item ini.

Semua flex-grow: 1 → lebar sama rata

grow:1

grow:1

Tengah flex-grow: 2 → dapat 2× ruang sisa

grow:1
grow:2

flex-shrink — Menyusut Saat Sempit

Menentukan seberapa besar item menyusut saat ruang tidak cukup.

flex-shrink: 0 → tidak menyusut (ukuran tetap)

shrink:1
shrink:0 (tetap)
shrink:1

flex — Shorthand

flex: grow shrink basis — menggabungkan ketiga properti.
Yang paling sering dipakai: flex: 1 (= grow:1, shrink:1, basis:0 → semua item lebar sama rata).

💡 Pola paling umum:
flex: 1 → semua item berbagi ruang sama rata
flex: 0 0 200px → ukuran tetap 200px, tidak tumbuh, tidak menyusut
flex: 1 1 200px → minimal 200px, bisa tumbuh dan menyusut

align-self — Override align-items per Item

Override align-items hanya untuk satu item tertentu.
