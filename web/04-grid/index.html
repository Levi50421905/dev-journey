4. grid-template-areas — Layout dengan Nama

Cara paling intuitif untuk mendefinisikan layout — beri nama setiap area, lalu gambarkan strukturnya seperti ASCII art. 

/*
  Container:
  "header  header  header"
  "sidebar main    main  "
  "footer  footer  footer"
*/
.container {
  display: grid;
  grid-template-columns: 200px 1fr 1fr;
  grid-template-rows: auto 1fr auto;
  grid-template-areas:
    "header  header  header"
    "sidebar main    main"
    "footer  footer  footer";
  gap: 12px;
  min-height: 300px;
}

/* Assign setiap item ke area-nya */
.header  { grid-area: header; }
.sidebar { grid-area: sidebar; }
.main    { grid-area: main; }
.footer  { grid-area: footer; }

💡 Tips: Pakai titik . untuk sel kosong:
"header header ." → kolom ketiga di baris header kosong

5. auto-fit & minmax() — Grid Responsif Otomatis

Kombinasi repeat(auto-fit, minmax()) adalah cara paling powerful untuk membuat grid yang otomatis menyesuaikan jumlah kolom berdasarkan lebar container — tanpa media query!

grid-template-columns: repeat(auto-fit, minmax(180px, 1fr))

→ Buat kolom sebanyak mungkin yang bisa muat
→ Setiap kolom minimal 180px, maksimal 1fr (sama rata)
→ Resize browser — kolom menyesuaikan otomatis!
